Classes to represent parsed keys, and utility to parse them. The various key objects can only represent what could possibly be valid keys.

There are four key types (NameLocatorKey, ContentLocatorKey, NameFullKey, ContentFullKey), and two general types (LocatorKey, FullKey), all of the general type Key.

A LocatorKey can be created from a FullKey, and is used to find the data that can be decoded using the FullKey.
