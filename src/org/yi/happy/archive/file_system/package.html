Features:

  • There is a functionally equivalent fake available.
  • An input stream for a file can be opened.
  • A file can be created from a byte array.
  • A file can be read into a byte array, possibly with a size limit.
  • The contents of a directory can be listed, sorted, and iterated.
  • A file can be tested to be a directory.
  • A file can be tested for existence.
  • A named child can be navigated to.
  • A directory can be created.
  • A file can be deleted.
  • A file can be renamed.
  • A file can be opened for random output.
